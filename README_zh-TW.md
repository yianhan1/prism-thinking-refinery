# 🔺 Prism Thinking Refinery

**一個為 OpenClaw 用戶打造的提升個人認知思維的技能。**

## 為什麼

AI 已經能幫你寫程式、寫信、做摘要。執行力不再是瓶頸——思維才是。

當 AI 接手了「做事」，真正拉開人與人差距的，是你提問的品質、看事情的廣度、和判斷的深度。但多數人一輩子只用一兩種思考模式，從來沒意識到自己的盲區在哪。

Prism Thinking Refinery 的出發點是：AI 能為你做的最有價值的事，不是幫你做事——而是讓你成為更好的思考者。它描繪你的思維模式、找出你的弱點、然後在最重要的地方推你成長。

## 運作原理

每個使用者都有一張 **Thinking Radar（思維雷達圖）**——一張蛛網狀的認知維度圖表。雷達不是靜態分數，它會隨著每次互動演化，根據你實際的思考表現調整 ±0.1 到 ±0.5。

## 八大認知維度

| 維度 | 核心問題 |
|------|---------|
| 🧱 **第一性原理** | 什麼是事實，什麼只是慣例？ |
| 🔄 **逆向思考** | 怎樣做一定會失敗？ |
| 👥 **利害關係人視角** | 其他人怎麼看這件事？ |
| 🕸️ **系統思維** | 這裡有什麼回饋迴路？ |
| ♟️ **賽局思維** | 如果我這樣做，對方會怎麼反應？ |
| 📜 **歷史類比** | 以前什麼時候發生過類似的事？ |
| 🌊 **二階效應** | 然後呢？再然後呢？ |
| 🔗 **跨領域連結** | 其他領域怎麼解決類似的問題？ |

維度可以自行新增（例如：倫理推理、美學判斷），系統會同等對待。

## 五大模式

| 模式 | 做什麼 |
|------|--------|
| 🔺 **稜鏡分析** | 用 3–5 個思考框架分析任何議題，優先針對你的弱項 |
| 🥊 **思維對練** | 真正有力度地挑戰你的想法——不是客氣的唱反調 |
| 📚 **精選推薦** | 跨領域深度閱讀推薦，根據你的成長方向量身打造 |
| 📓 **決策日誌** | 記錄預測，每月回顧，長期校準你的判斷力 |
| ✍️ **清晰寫作** | 把模糊的想法打磨成銳利的論述 |

## 自適應設計

技能根據你每個維度的分數調整互動方式：

- **低（1–3）：** 手把手帶你走過框架，給範例
- **中（4–6）：** 給你框架，讓你自己試，再補充
- **高（7–9）：** 直接在進階層次挑戰你的假設
- **精通（10）：** 請你反過來教——因為教學是最終的檢驗

## 隨時訓練 — 被動模式

Prism 不只是你打開來用的工具——它在日常對話中持續陪練：

| 機制 | 做什麼 |
|------|--------|
| 🪞 **思維回放** | 你在一般對話中做了判斷時，輕輕提示一個可以補充的思考維度 |
| 📰 **每日一問** | 推送時附上一個 30 秒的反思題，針對你最弱的維度 |
| 🏷️ **維度標記** | 當你自然展現強思維時，即時標注如 `（🔗 跨領域連結 +0.1）`——建立後設認知 |

## 安裝使用

1. 將技能放入你的 OpenClaw workspace（`skills/prism-thinking-refinery/`）
2. 執行 `node tools/prism-update.mjs --init` 建立資料檔案
3. 首次使用時，AI 會進行**校準對話**——情境式對話來建立你的初始雷達圖
4. 設定推送偏好（每天/每週推送精選內容）
5. AI 會在你的 `AGENTS.md` 加入被動訓練指令，讓認知教練在所有對話中運作
6. 開始用不同的方式思考

## 隱私

你的個人資料（`data/`——雷達分數、日誌、預測紀錄）預設在 `.gitignore` 中。技能框架可分享，你的思維剖析留在你手上。
